## Terminos Generales de Programacion
Los términos generales de programación abarcan los conceptos esenciales para crear software, incluyendo algoritmos (pasos lógicos), variables (almacenamiento de datos), sintaxis (reglas del lenguaje), funciones (código reutilizable) y estructuras de control (ciclos y condicionales). Estos fundamentos permiten a los desarrolladores escribir instrucciones, depurar código (bugs) y crear aplicaciones, sitios web o sistemas complejos
