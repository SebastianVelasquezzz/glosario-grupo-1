## Terminos de Hardware
### `CPU` — Unidad Central de Procesamiento
Es el cerebro del computador. Procesa todas las instrucciones
que le damos al sistema. Sin el, ninguna operacion es posible.
### `RAM` — Memoria de Acceso Aleatorio
Es la memoria temporal del computador. Guarda lo que estas
usando en este momento. Al apagar el computador, se borra.
### `SSD` — Disco de Estado Solido
Es donde se guarda la informacion de forma permanente.
Es mas rapido que un disco duro tradicional (HDD).
---
## Terminos de Software

### `Sistema Operativo (SO)` — Software principal del sistema
Es el programa que controla todo el computador. Administra el hardware y permite que otros programas funcionen. Sin el, el equipo no puede operar.

### `Aplicacion (App)` — Programa para el usuario
Es un software diseñado para realizar tareas especificas, como escribir, navegar en internet o editar fotos.

### `Programa` — Conjunto de instrucciones
Es un conjunto de instrucciones que le indican al computador que hacer. Todas las aplicaciones y sistemas operativos son programas.

### `Archivo` — Unidad de almacenamiento de datos
Es un conjunto de datos guardados en el computador, como documentos, imagenes o videos. Cada archivo tiene un nombre y una extension.

### `Software` — Parte logica del computador
Es el conjunto de todos los programas y sistemas que hacen funcionar el computador. No se puede tocar, pero es esencial para el uso del equipo.

## Terminos Generales de Programacion
Los términos de programación son conceptos fundamentales para crear software, incluyendo algoritmos (pasos lógicos), variables (almacenamiento de datos), sintaxis (reglas del lenguaje) y estructuras de control. Estos conceptos permiten a los desarrolladores escribir código, depurar errores y crear aplicaciones funcionales para el usuario.
